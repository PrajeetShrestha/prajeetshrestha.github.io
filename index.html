  • Microsoft Authentication Library (MSAL) Integration

3. Adkaddy - Ongoing development (iOS, Swift)
This application is a Smart Email client. My responsibility is on optimization of the codebase in this project.

One interesting thing that I have been involved in the project is of caching of emails. Emails contents are being fetched from the server as html string. We needed to make emails load faster just like other email client like gmail.

I have replaced the proxy server based caching to native platform based caching which had huge impact on processing and storage overhead on the serverside. It helped cut the cost of the company. And with this implementation scalability is lot more flexible in serverside. It also decreased the load time of emails by around 48% by in the appside.

I was only involved in iOS development for the project.

Prominent features of the application
  • Tricky UI
  • Content caching algorithm
  • Multiple analytics used
  • Gmail Integration
  • Distribution Method:Appstore and playstore.
4. Spiralogics Plus (Xamarin iOS, Android, C#, .net)
This application is internal application used by spiralogics employees. It is used to log and track hours in different projects of spiralogics. It pulls the ticket information from the azure boards and employee can log time spent on each ticket assigned to them. This is deployed to appstore with different public view.
This project was my first project on Xamarin and we followed MVVM pattern. Since xaml in Xamarin supported binding out of the box. And it was only natural to follow through MVVM pattern.

Prominent features of the application
  • MSAL Authentication
  • Charts
  • Distrubution method: Appstore and over the air for android.
5. Nepal Sambat (Xamarin iOS, Android, C#, .net)
This is Xamarin based project. This application is a Calendar for Nepal Sambat (or Newari Calendar) it also has conversion feature from, Nepal Sambat to A.D and B.S. This app consumes a proprietary DLL made in Spiralogics which has all the algorithm of calendrical calculations.

Note: This app was inaugurated by Hon. Prime Minister of Nepal Mr. Sher Bahadur Deuba on Nov 5, 2021.
This application have a lot of cultural significance and we got chance to collaborate with local community during the process. It has a potential to be of historical significance, since the algorithm used to calculate dates is very scientific and accurate. Nepal Samvat which was being used since 1143 years in Nepal now has a solid scientific foundation and documentation which previously lacked.

Prominent features of the application
  • DLL Integration
  • Custom Renderers
  • Bindable Layout for Grids
  • Distrubution method: Appstore and Playstore
