3. Adkaddy - Ongoing development (iOS, Swift)
This application is a Smart Email client. My responsibility is on optimization of the codebase in this project.

One interesting thing that I have been involved in the project is of caching of emails. Emails contents are being fetched from the server as html string. We needed to make emails load faster just like other email client like gmail.

I have replaced the proxy server based caching to native platform based caching which had huge impact on processing and storage overhead on the serverside. It helped cut the cost of the company. And with this implementation scalability is lot more flexible in serverside. It also decreased the load time of emails by around 48% by in the appside.

I was only involved in iOS development for the project.

Prominent features of the application
  • Custom charts
  • Real-time chat(twilio)
  • Geo fencing
  • SanDisk SDK Integration - We can download file from server and transfer it through a SanDisk flash drive connected to the ios device through this application
  • Microsoft Authentication Library (MSAL) Integration
